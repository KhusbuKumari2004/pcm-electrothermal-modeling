# Phase Change Memory (PCM) Electro-Thermal Simulation

## Overview

This project presents the electro-thermal modeling and simulation of a Phase Change Memory (PCM) device using MATLAB and COMSOL Multiphysics. The study investigates the thermal and electrical behavior of GST (Ge₂Sb₂Te₅) based PCM cells during phase transitions between amorphous and crystalline states.

The model analyzes temperature distribution, resistance variation, and switching characteristics under different voltage conditions. MATLAB is integrated with COMSOL to automate simulations and extract device characteristics efficiently.

## Objectives

- Model the electro-thermal behavior of PCM devices.
- Study phase transitions between amorphous and crystalline states.
- Analyze voltage-temperature characteristics.
- Analyze voltage-resistance characteristics.
- Investigate thermal effects caused by Joule heating.
- Automate simulation workflows using MATLAB and COMSOL integration.

## Results

The simulations demonstrate the influence of applied voltage on temperature distribution and resistance variation within PCM structures. The obtained results help in understanding switching behavior and thermal dynamics of PCM devices used in next-generation memory applications.
